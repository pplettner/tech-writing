App Integrations

Notion offers many external software plugins that allow you to automate and expedite your tasks.

 

The following tables list the integrations for software used here in WorkGeerz:

 

Software Development

App Features
Datadog
  • Manage and monitor Notion workspace activity in Cloud SIEM.
Github
  • Preview pull requests and issues in a page.
  • Import full repositories as Notion databases.
  • Link pull requests to individual Notion tasks.
Jira

  • Preview tickets in a page.

  • Import Jira projects and issues into a Notion database.
PagerDuty

  • Preview incident reports in a page.

 

Project Management

App Features
Asana

  • Import Asana tasks into a page.

  • Import Asana projects into a Notion database.
Google Drive

  • Preview Drive files in a Notion page.

  • Link Drive files in a Notion database.
Slack

  • Send Slack messages to a Notion database.

  • Create Notion tasks in Slack.

  • Enable Slack notifications for changes in Notion pages/databases.
Streamlit

  • Embed Streamlit apps on a page.
Tableau

  • Embed Tableau views on a page.

 

A full list of Notion integrations can be found here.
